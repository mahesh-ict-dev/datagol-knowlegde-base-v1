### Copying cURL for Workbook API Interaction

You can copy the cURL command to interact with the workbook using API requests. This command includes the necessary method, headers, and parameters for easy integration.

This cURL does not incorporate filters, sort conditions, or other view specific properties. To include such properties, copy the cURL command available for the specific view.

1.  On the Home page, from the left navigation panel, click Workspace and from the list of workspaces, open the workspace where the workbook that you want to edit is located.
    
2.  Click the Workbook tab and select the required workbook.
    
3.  Click the Settings button on the top and select Copy as cURL. The Copy cURL box is displayed with the cURL command.
    
4.  Click the copy icon to copy the cURL command.
