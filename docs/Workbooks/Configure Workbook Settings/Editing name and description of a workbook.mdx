### Editing name and description of a workbook

1.  On the Home page, from the left navigation panel, click Workspace and from the list of workspaces, open the workspace where the workbook that you want to edit is located.
    
2.  Click the Workbook tab and select the workbook that you want to edit.
    
3.  Click the Settings button on the top and select Edit Name and Description.
    
4.  Open image-20250428-074359.png
    
5.  Edit the name and description of the workbook and then click Save.
