## Creating workspace

1.  On the Home page, from the left navigation panel, click Workspace.
    
2.  In the Workspace page, click Create button. The Create Workspace page is displayed.
    
3.  Open image-20250512-080707.png
    
4.  In the Name field provide a name for the workspace and optionally provide a description in the Description text box.
    
5.  Specify the discovery settings of the Workspace.
    

-   Open to members: Company members can locate and join freely.
    
-   Members can join after approval: Company members can find the Workspace and request access, which the admin must then approve.
    
-   Hidden: Only members specifically invited by the admin can see and enter the Workspace.
    

7.  From the Customize Color section, you can select a profile color for the workspace.
    
8.  Click Create. The newly created workspace is displayed. This is an empty workspace, where you can add, import or publish workbooks.
